# 本站点的网页应用程序

## 导入

在第一次进入这个站点时，你的浏览器可能会跳出弹窗，提示是否要安装本站点的应用。

你也可以手动呼出这个下载界面，比如 Chrome 浏览器会在以下位置生成一个下载按钮：

![Chrome 安装 PWA](../../static/img/page-img/install-pwa-chrome.png)

Edge 的按钮则是这个样子：

![Edge 安装 PWA](../../static/img/page-img/install-pwa-edge.png)

点击按钮后，浏览器就会自动在你的设备中安装本站点的 PWA。

## 什么是 PWA

PWA 是渐进式网页应用程序的简称（Progressive Web App），它是基于 Web 技术而进行的拓展，可以让一个网站为用户提供一个简易的应用程序（类似小程序），从而为用户提供类似操作软件，而不是浏览网站的体验。

PWA 的行为与正常的应用程序类似：可以被安装到用户的设备上（手机、电脑等）、可以适应各种屏幕尺寸、可以在没有网络的情况下靠着缓存的数据脱机工作、可以与操作系统进行更深的交互（在用户的桌面弹出通知、在后台继续运行等）、可以被卸载和可以与其他应用程序互动等。

## 本站点的 PWA

本站点使用 Docusaurus 提供的插件进行了 PWA 配置，用户可以点击浏览器中的按钮下载本站点的 PWA。当处于无网络连接的环境时，本站点中的文章也可以被正常阅读。当站点的内容有更新时，页面右下角会出现**刷新**按钮，点击刷新即可同步到最新的站点内容。
